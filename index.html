Data Collection

We do not collect, store, or transmit any data whatsoever. This includes:

How It Works

All utilities on this page run entirely in your browser (client-side). When you provide your Butter CMS API token, it is used only to make direct API calls from your browser to Butter CMS's official API endpoints. Your token never passes through our servers because we don't have any servers.

The only external service this page interacts with is the official Butter CMS API (api.buttercms.com), and only when you explicitly use one of the utilities with your own API token.
